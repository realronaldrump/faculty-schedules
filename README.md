# Faculty Schedule Dashboard

This is a React application that allows you to find common availability between faculty members based on their teaching schedules.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (or [yarn](https://yarnpkg.com/)) installed on your machine.

### Installing

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```sh
   cd faculty-schedules
   ```
3. Install the dependencies:
   ```sh
   npm install
   ```

### Running the Application

To run the application in development mode, run the following command:

```sh
npm run dev
```

This will start the Vite development server and open the application in your default browser at `http://localhost:5173`.

### Building for Production

To create a production build of the application, run:

```sh
npm run build
```

This will create a `dist` directory with the optimized and minified production-ready files.

## Deployment

This application is ready to be deployed on services like Vercel or Netlify. Simply connect your Git repository to one of these services and they will automatically build and deploy the application.
